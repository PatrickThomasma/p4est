trans_sphere_to_cart(uvr)[source]#

Transforms coordinates from spherical to cartesian

Parameters:

uvr¶ (ndarray[(…, 3), dtype[floating]]) –

Spherical coordinates are (in order):

  • polar angle (aka. colatitude) \(\in [0, \pi]\)

  • azimuthal angle (aka. longitude) \(\in [-\pi, \pi]\)

  • radius

Returns:

ndarray[(…, 3), dtype[floating]] – Cartesian coordinates
