Hexahedral Mesh#

class HexMesh(verts, cells, vert_nodes=None, geoms=None, vert_geom=None)[source]#

Bases: object

Base container for hexahedral mesh

Parameters:

  • verts¶ (Vertices ⊆ ndarray[(NV, 3), dtype[floating]]) – Position of each vertex. (AKA p8est_connectivity_t.vertices) Indexing is [vertex, (x,y,z)]

  • cells¶ (Cells ⊆ ndarray[(NC, 2, 2, 2), dtype[integer]]) – Mapping of hexahedral cells to the indices of their 8 vertices. (AKA p8est_connectivity_t.tree_to_vertex)

  • vert_nodes¶ (VertNodes ⊆ ndarray[(NV,), dtype[integer]], optional) – The topological node associated with each vertex, causing cells to be connected by having vertices associated with the same node in addition to directly sharing vertices. A value of -1 is used to indicate independent vertices. If not given, each vertex is assumed to be independent, and cells are only connected by shared vertices.

  • geoms¶ (Sequence[HexGeometry], optional) – The available geometries that may be referenced by ‘vert_geom’. (default: [HexLinear])

  • vert_geom¶ (VertGeom ⊆ ndarray[(NV,), dtype[integer]], optional) – Indices into ‘geoms’ to get the geometry associated with each vertex. (default: zeros(NV))

Jacobian of the absolute coordinates w.r.t local coordinates

\[\begin{split}\ranktwo{J}_\rankone{r} = \nabla_{\rankone{q}} \rankone{r} = \begin{bmatrix} \frac{\partial x}{\partial q_0} & \frac{\partial x}{\partial q_1} & \frac{\partial x}{\partial q_2} \\ \frac{\partial y}{\partial q_0} & \frac{\partial y}{\partial q_1} & \frac{\partial y}{\partial q_2} \\ \frac{\partial z}{\partial q_0} & \frac{\partial z}{\partial q_1} & \frac{\partial z}{\partial q_2} \end{bmatrix}\end{split}\]
